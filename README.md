# fictional-city_map
I’m making a map of a fictional city that I will later create buildings for on the software called QGIS. This repo contains both my QGIS files and my Figma work that I used to create a website mockup for my QGIS files: https://www.figma.com/proto/ibXAInaHiCfNgEBLMBl9UG/Senior-Project?node-id=3-5&t=CFfa0CqHDBPDOd73-1

WHAT IS GIS (Geographic Information Systems):
Basically what Google Maps does and what urban planners work with to build data on maps. In this project, QGIS is one software that can build the maps for you kind of like how CAD software works. I think it's super interesting because there are endless ways to customize a map (urban heat islands, water pollution, traffic, etc etc).
